# ⚡ Reality.exe

> “You thought you were in control… but you stayed.”

---

## 🧠 About

**Reality.exe** is a deceptive, interactive web experience that challenges user perception through psychological tricks, gamified interactions, and dynamic UI behavior.

Instead of traditional navigation, the website unfolds as a **continuous journey**, where every action leads forward — but not always as expected.

---

## 🎮 Features

* 🎭 Illusion of choice (all paths lead forward)
* 🎮 Mini-games:

  * Follow-the-dot challenge
  * Patience test (do nothing)
  * Memory manipulation game
  * Control inversion game
* 🧩 Riddles & hidden interactions
* 🌌 Anti-gravity UI (floating elements, parallax effects)
* 🎨 Dark futuristic design (neon blue + black theme)
* 🎵 Progressive sound (calm → glitch → intense)

---

## 💀 Unique Twist

A memory game that **changes the sequence after the user answers**, creating a moment of confusion and surprise.
